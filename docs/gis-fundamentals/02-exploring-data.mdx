**Key Terms & Definitions:**
1. Watershed - A land-based area that drains to a single point
2. Definition Query - A way to limit what features are displayed based on attribute values
3. Scale-dependent rendering - Controls when layers are visible based on zoom level
4. Layer Properties - Dialog box containing settings and information about a GIS layer
5. Unique Values renderer - Draws categories using unique values from data fields
6. Symbology - Visual representation of features on a map

**Important Examples:**
1. Data Types shown:
  - NHDFlowline (Rivers)
  - NHDWaterbody (Lakes)
  - WBD_Subwatershed (Watersheds)

2. Attribute Example:
  - TotDA (Total Drainage Area in square kilometers)
  - HUC_8 (Watershed identifier)

**Key Steps Demonstrated:**
1. Renaming Layers:
  - Double click slowly on layer name
  - Use F2 key
  - Right-click and select Properties

2. Changing Layer Symbology:
  - Click on color patch
  - Use symbology pane
  - Select predefined styles or customize colors

3. Setting Scale-Dependent Rendering:
  - Right-click layer → Properties
  - Set "do not show layer when zoomed" values
  - Specify scale values (e.g., 1:500,000)

4. Creating Definition Queries:
  - Access layer properties
  - Click "New Definition Query"
  - Add clause (e.g., TotDA > 5)

5. Adding Labels:
  - Right-click layer → Select label
  - Double-click field name for label text
  - Customize label appearance (font, size, color)
